# Tony Baldwin's Claude plugins

A Claude Code plugin marketplace.

## Install

```
/plugin marketplace add mr-fumanchu/claude-plugins
```

Then install a plugin:

```
/plugin install smoke-signal@claude-plugins
```

Restart Claude Code so hooks load.

## Plugins

- **smoke-signal** — always-on status readout: active project, live model and reasoning effort, session and project cost, and context-window usage, with a red compaction alert plus an animated fire-and-smoke widget at 75%. If you already run Smoke Signal manually, uninstall it first so the hooks do not double.
